# Muzikbox 001

Commission prototype.

## Dimensions

*Box*: `154mm x 127mm x 45mm`

> Rough conversions to imperial
> ```
> 153mm   == 6in
> 127mm   == 5in
> 51mm    == 2in
> 19.05mm == 3/4in
> 3.175mm == 1/8in
> ```

- 50 units * top + bottom == 100
- top/bottom: `154*127*100 == 1955800mm^2` (roughly 2m^2)
- standard ply sheet: `4x8ft == 32ft^2 == 2.9729m^2` (roughly 3m^2)
